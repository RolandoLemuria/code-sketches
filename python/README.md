# Python Code Sketches

This directory contains quick sketches and examples written in Python. These sketches are intended to demonstrate various programming concepts, algorithms, or functionalities in a concise manner.

## Usage

To run any of the sketches, ensure you have Python installed on your machine. You can execute the scripts using the following command in your terminal:

```
python <script_name>.py
```
