# TypeScript Code Sketches

This directory contains various sketches written in TypeScript. Each sketch serves as a quick example or demonstration of TypeScript features, concepts, or libraries.

## Getting Started

To run the TypeScript sketches, ensure you have Node.js and TypeScript installed. You can install TypeScript globally using npm:

```
npm install -g typescript
```

## Usage

1. Navigate to the TypeScript directory:
   ```
   cd code-sketches/typescript
   ```

2. Compile the TypeScript files:
   ```
   tsc <filename>.ts
   ```

3. Run the compiled JavaScript:
   ```
   node <filename>.js
   ```
