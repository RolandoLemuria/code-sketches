# C++ Sketches

This directory contains various sketches written in C++. Each sketch serves as a quick example or demonstration of different C++ features, concepts, or algorithms.

## Usage

To run the sketches, ensure you have a C++ compiler installed. You can compile and execute the sketches using the following commands:

```bash
g++ sketch_name.cpp -o sketch_name
./sketch_name
```
