# C# Sketches

This directory contains sketches written in C#. Each sketch is a quick example or demonstration of a specific concept or functionality in C#. 

## Usage

To run the sketches, ensure you have the .NET SDK installed. You can create a new console application for each sketch or run them in an interactive environment like LINQPad.

## Examples

- **Basic Syntax**: A simple demonstration of C# syntax.
- **Data Structures**: Examples of using lists, dictionaries, and other data structures.
- **LINQ Queries**: Showcasing the power of LINQ for data manipulation.

Feel free to contribute your own sketches or modify existing ones!
