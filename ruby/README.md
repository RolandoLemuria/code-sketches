# Ruby Code Sketches

This directory contains various sketches written in Ruby. Each sketch serves as a quick example or demonstration of Ruby's features and capabilities. 

## Usage

To run a Ruby sketch, ensure you have Ruby installed on your machine. You can execute a sketch by navigating to its directory in your terminal and running:

```
ruby <sketch_file>.rb
```

## Contributing

Feel free to add your own Ruby sketches! Just create a new Ruby file in this directory and document its purpose in this README.
