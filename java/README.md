# Java Code Sketches

This directory contains various sketches and examples written in Java. Each sketch is designed to demonstrate specific concepts, algorithms, or features of the Java programming language.

## Usage

To run the Java sketches, ensure you have the Java Development Kit (JDK) installed on your machine. You can compile and run the sketches using the following commands:

1. Navigate to the sketch directory.
2. Compile the Java file:
   ```
   javac SketchName.java
   ```
3. Run the compiled Java program:
   ```
   java SketchName
   ```
